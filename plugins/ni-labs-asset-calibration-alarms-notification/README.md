# Asset Calibration Alarms and Notification

This plugin packages the public NI SystemLink example notebook as a Plugin Manager notebook package.

Source notebook:
- https://github.com/ni/systemlink-enterprise-examples/blob/main/examples/Alarms%20and%20Notification%20Examples/Asset%20Calibration%20Alarms%20and%20Notification.ipynb

The payload under `notebook/` is copied into `dist/notebook/` and then packaged as a `.nipkg` with `XB-Plugin: notebook` plus Plugin Manager catalog metadata.
